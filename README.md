# Arulmigu Kallichi Amman Temple Management System

Upload ALL files in this package to your GitHub repository.

## Files:
- `index.html` — Main temple app (open directly in browser or iPhone Safari)
- `manifest.json` — iPhone PWA support
- `sw.js` — Offline support
- `icon-192.png` / `icon-512.png` — App icons
- `.github/workflows/build.yml` — Auto-builds Android APK

## How to get APK:
1. Upload all files to GitHub repo (Public)
2. Go to Actions tab → wait for green tick
3. Download APK from Artifacts section

## How to open on iPhone:
1. Enable GitHub Pages (Settings → Pages → main branch)
2. Open URL in Safari → Share → Add to Home Screen
